Robot Moderation

Robot Moderation is a simple Moderation made by Dan.#2518 and Mr Robot#8472, and is made for the Mr Robot Community Discord Server.

Command List

Moderation Commands

  • ban - Bans a mentioned user. !ban [user] [reason]
  • kick - Kicks a mentioned user. !kick [user] [reason]
  • purge - Purges a certain amount of messages. !purge [amount of messages]

Utility Commands

  • about - Posts an embed saying some information about Robot Moderation. !about
  • announce - Posts an embed with whatever text you want. !announce [text]
  • avatar - Posts an embed with your avatar URL. !avatar
  • github - Posts a link to the GitHub Repository for Robot Moderation. !github
  • help - DMs you an embed listing all the commands in Robot Moderation. !help
  • invite - Posts an embed with the invite link to add Robot Moderation to your server. !invite
  • ping - Pings your connection to the bot with a milliseconds value. !ping
  • reload - Reloads a certain command, without having to restart the bot. !reload [command]
